Disk Management Commands

Disk management in Linux involves handling partitions, mounting file systems, checking disk usage, and managing storage devices.

Partition Management

  • fdisk -l - list all partitions
  • fdisk /dev/sdX - modify disk partitions
  • parted /dev/sdX - advanced partition management
  • lsblk - display block devices

File System Management

  • mkfs.ext4 /dev/sdX1 - create an ext4 file system
  • mkfs.xfs /dev/sdX1 - create an XFS file system
  • mount /dev/sdX1 /mnt - mount a file system
  • umount /mnt - unmount a file system

Disk Usage & Monitoring

  • df -h - display disk space usage
  • du -sh /path/to/directory - show directory size
  • iostat - report CPU and disk I/O statistics
  • iotop - show real-time disk usage by processes

Checking & Repairing File Systems

  • fsck /dev/sdX1 - check and repair file system
  • tune2fs -l /dev/sdX1 - show ext2/ext3/ext4 file system details
  • e2fsck -f /dev/sdX1 - force a file system check

Swap Management

  • swapon -s - display swap usage
  • mkswap /dev/sdX2 - create a swap partition
  • swapon /dev/sdX2 - enable swap partition
  • swapoff /dev/sdX2 - disable swap partition
